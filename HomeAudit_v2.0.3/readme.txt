Описание исходника:
Пример создания базы данных по учету доходов, расходов, долгов, командировок.
Не требует инсталляции (один екзешник и один файл базы данных *.fdb).
Необходима предустановка бесплатной СУБД FireBird 2.5.1 (http://www.firebirdsql.org/en/firebird-2-5-1/).
Дополнительные библиотеки: EhLib, RxLib, ElTree, IBO, Toolbar97, LMD, EasyGraph.
